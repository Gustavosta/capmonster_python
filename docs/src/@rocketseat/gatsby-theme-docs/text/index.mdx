# Introduction

Capmonster is a cloud service for automatic recognition of reCAPTCHA, hCaptcha and other types of captchas.
This is the documentation of [Capmonster.cloud](https://capmonster.cloud) Python3 package.

## Available Captcha Types

- Image to Text
- ReCaptcha V2
- ReCaptcha V2 Enterprise
- FunCaptcha
- HCaptcha
- ReCaptcha V3
- GeeTest
